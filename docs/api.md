# API Overview

The control-plane API is now owned by `langbridge-cloud/`.

Use:

- `../langbridge-cloud/docs/api.md`
- `../langbridge-cloud/docs/insomnia/langbridge_collection.yml`
- `../langbridge-cloud/docs/postman/langbridge_collection.json`

Regenerate API collections from the cloud repo with:

```bash
cd ../langbridge-cloud
python scripts/generate_api_collections.py
```
